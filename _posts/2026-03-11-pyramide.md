# Pyramide

## Entstehung

Das muss so 1993 oder 1994 in Braunschweig gewesen sein. Ein paar Azubis von der DLR saßen in einer Kneipe und hatten Würfel. Ich weiß nicht mehr, wer genau dabei gewesen ist. Jedenfalls war auch Alkohol im Spiel.

## Zutaten

Es werden benötigt, sechs Würfel und Stift und Papier zum Notieren der Punkte.

## Ablauf

Man kann es alleine oder mit vielen spielen.

Es werden sechs Runden gespielt. Jeder Spieler hat in jeder Runde drei Würfe, um möglichst viele Punkte zu erreichen.

Ein Spieler würfelt zunächst mit allen sechs Würfeln.    
Man versucht, aus den Würfeln eine Pyramide zu legen (man ordnet die Würfel flach auf dem Tisch an, stapeln in die Höhe wäre zu schwierig).    
Das sieht dann so aus:
```
    +---+
    |   |
    |   |
    |   |
  +---+---+
  |   |   |
  |   |   |
  |   |   |
+---+---+---+
|   |   |   |
|   |   |   |
|   |   |   |
+---+---+---+
```

Folgende Regeln müssen dabei eingehalten werden:

* der Wert eines Würfels Z über zwei anderen Würfeln X und Y muss Y-X+1 sein (wobei Y größer oder gleich X ist). Beispiele:
```
    +---+
    |   |
    | 2 |
    |   |
  +---+---+
  |   |   |
  | 6 | 5 |
  |   |   |
  +---+---+ 2 = 6-5 +1
```
```
    +---+
    |   |
    | 1 |
    |   |
  +---+---+
  |   |   |
  | 4 | 4 |
  |   |   |
  +---+---+ 1 = 4-4 +1
```
* eine vollständige große Pyramide besteht aus sechs Würfeln. Beispiel:
```
....+---+
....|   |
....| 2 |
....|   |
..+---+---+
..|   |   |
..| 3 | 2 |
..|   |   |
+---+---+---+
|   |   |   |
| 4 | 6 | 5 |
|   |   |   |
+---+---+---+
```
* eine halbe Pyramide besteht aus fünf Würfeln:
```
..+---+---+
..|   |   |
..| 3 | 2 |
..|   |   |
+---+---+---+
|   |   |   |
| 4 | 6 | 5 |
|   |   |   |
+---+---+---+
```
* eine kleine Pyramide besteht aus drei Würfeln
```
..+---+
..|   |
..| 2 |
..|   |
+---+---+
|   |   |
| 6 | 5 |
|   |   |
+---+---+
```

Der Spieler hat drei Versuche, um eine möglichst große Pyramide mit möglichst hohem Wert zu erwürfeln. Bei jedem Versuch darf er entscheiden, wieviele Würfel rausgelegt werden sollen. Es dürfen auch bereits herausgelegte Würfel erneut mitgewürfelt werden.    
Gebaut wird nach drei Versuchen.

Punkte gibt es nach folgenden Regeln:

* bei einer großen Pyramide zählen die Punkte dreifach, bei einer halben doppelt, bei einer kleinen einfach
* nicht verbaute Würfel sind negative Punkte
* grundsätzlich gilt die Gesamtsumme der Augen aller Würfel in der Pyramide, mit folgender Ausnahme für große und halbe Pyramiden:
* wenn die Würfel in der untersten Reihe alle den gleichen Wert haben (dann müssen die Würfel darüber Einsen sein), hat die Pyramide folgenden Wert:
  * bei 2en: 20
  * bei 3en: 30
  * bei 4en: 40
  * bei 5en: 50
  * bei 6en: 60
  * bei 1en: 100

  Beispiele:

* eine halbe Pyramide und eine nicht verbaute 6
```
..+---+---+
..|   |   |
..| 3 | 2 |
..|   |   |
+---+---+---+ +---+
|   |   |   | |   |
| 4 | 6 | 5 | | 6 |
|   |   |   | |   |
+---+---+---+ +---+
```
(4+6+5+3+2)*2 -6 = 34
* eine große Pyramide
```
....+---+
....|   |
....| 2 |
....|   |
..+---+---+
..|   |   |
..| 3 | 2 |
..|   |   |
+---+---+---+
|   |   |   |
| 4 | 6 | 5 |
|   |   |   |
+---+---+---+
```
(4+6+5+3+2+2)*3 = 66
* eine halbe Pyramide mit 5en und einer nicht verbauten 6
```
..+---+---+
..|   |   |
..| 1 | 1 |
..|   |   |
+---+---+---+ +---+
|   |   |   | |   |
| 5 | 5 | 5 | | 6 |
|   |   |   | |   |
+---+---+---+ +---+
```
50*2 -6 = 94
* eine große Pyramide mit 6en
```
....+---+
....|   |
....| 1 |
....|   |
..+---+---+
..|   |   |
..| 1 | 1 |
..|   |   |
+---+---+---+
|   |   |   |
| 6 | 6 | 6 |
|   |   |   |
+---+---+---+
```
60*3 = 180
* eine halbe Pyramide mit 1en und einer nicht verbauten 2
```
..+---+---+
..|   |   |
..| 1 | 1 |
..|   |   |
+---+---+---+ +---+
|   |   |   | |   |
| 1 | 1 | 1 | | 2 |
|   |   |   | |   |
+---+---+---+ +---+
```
100*2 -2 = 198
* die größte Pyramide
```
....+---+
....|   |
....| 1 |
....|   |
..+---+---+
..|   |   |
..| 1 | 1 |
..|   |   |
+---+---+---+
|   |   |   |
| 1 | 1 | 1 |
|   |   |   |
+---+---+---+
```
100*3 = 300

Der Spieler notiert seine Punktzahl für diese Runde auf einem Spielblatt der folgenden Art
```

    ___           x3 = ___

  ___ ___   Summe x2 = ___

___ ___ ___ Summe x1 = ___
      Summe gesamt   = ___
```
Ein Eintrag in der Spitze der Pyramide zählt dreifach, in der Mitte doppelt und unten einfach. In jeder Runde wählt der Spieler, wo die Punktzahl der Runde eingetragen werden soll. Was einmal eingetragen ist, darf nicht mehr geändert werden.

Beispiel nach sechs Runden:
```
     64           x3 = 192
    ___                ___
   66  54   Summe x2 = 240
  ___ ___              ___
 32  24  -4 Summe x1 =  54
___ ___ ___            ___
      Summe gesamt   = 486
                       ___
```

Es gewinnt, wer nach sechs Runden die meisten Punkte hat.
